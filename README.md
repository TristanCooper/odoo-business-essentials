# Weekly Orders Kanban (Odoo 17)
Odoo addon that provides a weekly-style customer board on Contacts.

- Shows flagged customers on a Kanban grouped by weekly stage (Schedule, Monâ€“Fri, Order Received).
- Whole card opens the Partner; email/phone shown when present.
- Dragging to a weekday creates a call activity for that day of the current week.

Requirements: Odoo 17, base, mail, sale_management.
Install: copy `weekly_orders_kanban` into your addons path and Update Apps List.
